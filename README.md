# VR Escape Room

An interactive **Virtual Reality Escape Room** experience designed to simulate real-world puzzle-solving in an immersive 3D environment. This project demonstrates the integration of game development, user interaction, and problem-solving mechanics within a virtual setting.

---

## 🚀 Overview

The VR Escape Room project is a simulation-based application where users navigate a virtual environment, interact with objects, and solve puzzles to progress and escape the room. The goal is to create an engaging and immersive experience that combines logic, exploration, and user interaction.

This project showcases skills in **game design, 3D environment development, and interactive system implementation**.

---

## 🧠 Tech Stack

- **Game Engine:** Unity  
- **Language:** C#  
- **Platform:** Virtual Reality / Desktop (depending on setup)  
- **Tools:**  
  - Unity Editor  
  - Visual Studio / VS Code  
  - 3D Assets & Scene Design  

---

## 🎯 Features

- Interactive 3D environment  
- Puzzle-based gameplay  
- Object interaction (pick, inspect, use)  
- Immersive VR experience  
- Logical progression system  
- User-driven exploration

## 🎮 How It Works

1. Player enters the virtual escape room  
2. Explores the environment  
3. Interacts with objects and clues  
4. Solves puzzles step-by-step  
5. Unlocks progression mechanisms  
6. Escapes the room upon completing all challenges
